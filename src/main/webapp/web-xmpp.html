Web XMPP

A Vysper + Strophe.js powered web chat

This chat automatically connects to the XMPP server and allow to all the users who opened the page to write message in a multi-user chat room.

A user name can be specified by loading the page with the username parameter. (e.g. http://.../web-xmpp/?username=foo)

The user will automatically quit the chat room once the page is closed.

You can send private messages to a given user by using the <- operator. For example user@web-xmpp <- Hello will send a private message to user.
